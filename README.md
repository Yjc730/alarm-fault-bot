# Alarm-Fault 對話機器人

## 專案簡介

Alarm-Fault 對話機器人是針對基地台維運情境設計的告警查詢工具，協助使用者快速查詢告警代碼、故障資訊與相關說明。

本專案將傳統文件查找流程轉換為對話式查詢介面，讓使用者可以透過輸入告警代碼或關鍵字，快速取得相關資訊，提升查詢便利性。

## 開發動機

在基地台維運工作中，告警代碼與故障說明通常分散在不同文件或資料表中。若使用者需要人工翻閱資料，會增加查找時間與操作成本。

因此本專案希望建立一個更直覺的查詢工具，讓使用者能以對話方式快速查找 Alarm / Fault 相關資訊，並可進一步整合至企業智能助理中。

## 主要功能

- 告警代碼查詢
- 故障說明查詢
- 關鍵字搜尋
- 對話式查詢介面
- 查詢結果整理與顯示
- 可整合至企業智能助理 Beta

## 技術架構

- Frontend：HTML、CSS、JavaScript
- Data：JSON 告警資料
- Query Logic：關鍵字比對、資料查詢邏輯
- Interface：對話式查詢介面
- Integration：可整合至企業智能助理系統

## 系統流程

```text
使用者輸入告警代碼或關鍵字
        ↓
前端接收查詢內容
        ↓
系統比對 JSON 告警資料
        ↓
整理對應的告警說明與故障資訊
        ↓
回傳查詢結果並顯示於對話介面
